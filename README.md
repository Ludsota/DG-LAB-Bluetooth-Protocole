# DG-LAB Coyote V3.0 & PawPrints ‚Äì Local WebBluetooth Toolkit

This repository collects and documents my **independent research** on connecting **DG-LAB Coyote V3.0** and **DG-LAB PawPrints** directly from a **local computer** without relying on the official app or third-party apps (xToys).

The goal is to make it possible to control devices using **Web Bluetooth**, through a browser such as **Chrome**, in a fully standalone setup.

---

## ‚ú® What is inside?

- Practical notes about the BLE protocol (services, characteristics, payloads)
- Web-Bluetooth test pages (local, static HTML)
- Minimal JavaScript examples
- Device exploration tools
- Experimental UI controls

Everything runs **locally** in the browser.  
No external servers, no external platform, no uploads, no cloud.
